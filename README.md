# VK Repair Bot

## Setup Instructions

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/artemkanote/vk-repair-bot.git
   cd vk-repair-bot
   ```

2. **Install Dependencies**:
   Make sure you have Python installed, then use:
   ```bash
   pip install -r requirements.txt
   ```

3. **Configuration**:
   - Create a configuration file with your settings.
   - Rename `config.example.json` to `config.json` and fill in the required fields.

4. **Run the Application**:
   ```bash
   python main.py
   ```
